<p align="center">
<b>MAPLE</b>: mapping and visualization of similar primer and adapter sequences</p>
<br>

Installation:
<pre>
git clone https://github.com/pratas/maple.git
cd maple/
</pre>

Run synthetic:
<pre>
chmod +x Synthetic.sh.sh
./Synthetic.sh
</pre>

Run directory:
<pre>
chmod +x Maple_dir.sh.sh
./Maple_dir.sh
</pre>
